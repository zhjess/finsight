# Finsight

![Finsight](./docs/Cover-Finsight.png)

## Tech stack

Language: TypeScript

Front End: React, Vite, Redux, Material UI, Recharts, Regression

Back End: ExpressJS, NodeJS, Prisma

Database: PostegreSQL

Hosting: Vercel
